Detailed Description

Method Documentation

checkDeviceID( connection)

Checks that the device is a SAMx7.

Reads CHIPID_CIDR register using the given connection and display a warning if its value does not match the expected value for SAMx7.


initialize( connection)

Initialize the SAMV7 device using the given connection.

This method calls checkDeviceID.
