Detailed Description

By default, no configuration values are set.

SPI Configuration

The following SPI configurations are supported:

InstancePeripheralI/O SetChip Selects
0SPI01, 20, 1, 2, 3
1SPI11, 20, 1, 2, 3
1SPI130, 1, 2
2FLEXCOM010, 1
3FLEXCOM110, 1
4FLEXCOM21, 20, 1
5FLEXCOM31, 2, 30, 1
6FLEXCOM41, 2, 30, 1

QuadSPI Configuration

The following QuadSPI configurations are supported:

InstancePeripheralI/O Set
0QSPI01, 2, 3
1QSPI11, 2, 3

NAND Configuration

The following NAND configurations are supported:

I/O SetBus Width
1, 28-bit, 16-bit

SDMMC Configuration

The following SDMMC configurations are supported:

InstancePeripheralBus Width
0SDMMC01-bit, 4-bit, 8-bit
1SDMMC11-bit, 4-bit
