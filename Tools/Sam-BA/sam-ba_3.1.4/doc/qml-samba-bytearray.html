Detailed Description

Property Documentation

length : int

the byte array length


Method Documentation

void append(ByteArray other)

Appends the content of byte array other to the current data.


int at(int idx)

Returns the byte at index idx.


var compare(ByteArray other)

Compare the content of byte array other to the current data.

Returns the offset of the first difference, or no value (undefined) if no difference was found.


ByteArray mid(int index, int len)

Returns a byte array containing len bytes from index index.


void pad(int count, int value)

Pad the current data up to count bytes.

Added bytes are initialized with value. If the array was longer that count bytes, it will be truncated.


void prepend(ByteArray other)

Prepend the content of byte array other in front of the current data.


bool readFile(string fileName)

Read the contents of file named fileName into the byte array.

Returns true on success, false otherwise


bool readUrl(string fileUrl)

Read the contents pointed to by URL fileUrl into the byte array.

Returns true on success, false otherwise


bool writeFile(string fileName)

Write the contents of the byte array into the file fileName.

Returns true on success, false otherwise
