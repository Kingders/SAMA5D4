Detailed Description

Method Documentation

ByteArray createByteArray(int length)

Create a new empty ByteArray of size length bytes.

Returns the newly created ByteArray.


string hex(number value, number digits)

Convert value to a hexadecimal string with digits.

Returns the hexadecimal string


void msleep(int msecs)

Pause the current thread for msecs milliseconds.


string parseInteger(string text)

Convert decimal or hexadecimal string to a number.

Returns the parsed number, or Number.NaN if the string could not be parsed.


ByteArray readFile(string fileName)

Read the contents of file fileName into a new ByteArray.

Returns the newly created ByteArray, or undefined if an error occurred.


ByteArray readUrl(string fileUrl)

Read the contents of URL fileUrl into a new ByteArray.

Returns the newly created ByteArray, or undefined if an error occurred.


void sleep(int secs)

Pause the current thread for secs seconds.


void usleep(int usecs)

Pause the current thread for usecs microseconds.


bool writeFile(string fileName, ByteArray data)

Write the byte array data into file fileName.

Returns true on success, false otherwise.
